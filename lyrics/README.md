#

`python3 -m venv .venv`

Activate with `.venv/Scripts/activate` or `.venv\Scripts\activate.bat` or `.venv\Scripts\Activate.ps1`

Install:
`pip install -y --upgrade pip`
`pip install -y openai-whisper ffmpeg-python`
Note that ffmpeg need to be installed and in the PATH.

Run:
`python mp3_to_lrc.py ..\ghost.mp3`
